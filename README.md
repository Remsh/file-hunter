# file-hunter
To inspect which directory is increasing rapadliy in Linux/Unix

Usage:
    chmod +x main
    
    ./main /home   ## /home is the dir need inspected
    
    ./main         ## default dir is /



This will show the result after 2 miniutes, any big file changes will show in the Result.
